# fsdsem3 - Full Stack Development (Semester 3)

Welcome to the **Full Stack Development (Semester 3)** repository. This repository contains course materials, practical lab assignments, projects, and codebase references developed during Semester 3.

---

## 🚀 Overview

This repository is structured to organize all assignments, laboratory work, and full-stack projects covering modern web technologies.

- **Frontend Development:** HTML5, CSS3, JavaScript (ES6+), React.js
- **Backend Development:** Node.js, Express.js
- **Database & Storage:** MongoDB / SQL
- **Tooling & Version Control:** Git, GitHub, npm, Vite

## 🛠️ Getting Started

### Prerequisites

Ensure you have the following tools installed locally:

- [Node.js](https://nodejs.org/) (v18.x or higher)
- [Git](https://git-scm.com/)

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sparsh101sparsh/fsdsem3.git
   cd fsdsem3
   ```

2. **Navigate to the desired directory:**
   ```bash
   cd frontend   # or cd backend / labs
   ```

3. **Install dependencies (if applicable):**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
